= SESSION_CHECKLIST — graphify-gradle

== Hook d'Ouverture — Avant de commencer une session

. [ ] Lire `PROMPT_REPRISE.adoc` — contexte de la session precedente
. [ ] Lire `AGENT.adoc` — regles absolues
. [ ] Lire `.agents/INDEX.adoc` — roadmap, etat des EPICs, portefeuille
. [ ] Verifier l'etat git (`git status`) — branche, fichiers modifies
. [ ] Verifier que le build compile (`./gradlew compileKotlin`)
. [ ] Si le code source a ete modifie : `./gradlew publishToMavenLocal`

== Pendant la session

. [ ] Respecter la Regle 0 — publishToMavenLocal OBLIGATOIRE apres modif
. [ ] Respecter la Regle 1 — AUCUN commit sans permission explicite
. [ ] Respecter la Regle 1b — backup avant modif fichiers config
. [ ] Respecter la Regle 2 — PAS de tests en fin de session sans permission
. [ ] Verifier que chaque modification compile avant de continuer

== Hook de Fermeture — Procedure 6 etapes

. [ ] 1. Rapport de session → `.agents/sessions/NNN-sujet.adoc`
. [ ] 2. Mise a jour `PROMPT_REPRISE.adoc` — contexte reprise N+1
. [ ] 3. Mise a jour `INDEX.adoc` — numero de session, roadmap, statut EPICs
. [ ] 4. Mise a jour `SESSIONS_HISTORY.adoc` — ajouter entree
. [ ] 5. Mise a jour `TEST_COVERAGE_ANALYSIS.adoc` (si tests modifies)
. [ ] 6. Mise a jour `COMPLETED_TASKS_ARCHIVE.adoc` — taches terminees

== Commandes de Verification

[source,shell]
----
# Verifier l'etat git
git status

# Verifier l'absence de secrets
grep -r "TOKEN\|SECRET\|PASSWORD\|API_KEY" --include="*.kts" --include="*.properties" --include="*.yml"

# Verifier que le build compile
./gradlew compileKotlin compileTestKotlin

# Publier en local si modif
./gradlew publishToMavenLocal
----
